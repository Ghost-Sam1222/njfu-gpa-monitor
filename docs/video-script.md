## 录制规范

- 不展示真实学号、密码、Cookie、Webhook、邮箱授权码或 Device Key。
- Pages、Codespaces、设置页、Actions 和通知示例都要出现，避免误导用户只能使用 Bark。
- 镜头切换间隔保持在 0.5 至 1 秒；Codespace 启动等待使用压缩画面和文字说明。
- 不承诺绝对免维护。学校登录页或 GitHub 规则变化时，由用户在 Actions 页面手动确认上游更新，程序不会静默替换可接触 Secrets 的代码。
