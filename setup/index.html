正在连接 GitHub…

1. 教务账号

使用 Cookie 登录（高级）

2. 检查频率

推荐选择一个预计所有成绩已公布的停止日期；当天会发送当前成绩单。达到完成条件后只发送一次提醒，但仍会检查到停止日期，以便发现成绩更正。

3. 通知渠道

只需配置一种。不知道怎么选？打开逐步教程

Bark（iPhone / iPad）
飞书、钉钉、企业微信与 Slack
Telegram 与 ntfy

先给自己的机器人发送一条消息，再点击获取；Token 不会进入浏览器地址栏。

邮件（所有设备）

这里只使用 SMTP 发信，不需要 IMAP。QQ、163 请填写客户端授权码，不要填写邮箱登录密码。

通用 Webhook

每月保活始终开启，只更新一个公开心跳文件。程序更新不会静默安装，需要你在 Actions 中手动确认。

敏感信息只会写入 GitHub Secrets，空白项不会覆盖已有值。
